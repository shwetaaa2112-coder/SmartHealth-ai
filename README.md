# SmartHealth AI Dashboard 🏥

SmartHealth AI ek modern aur responsive web application hai jo users ko unka BMI aur health analytics track karne mein madad karta hai.

## ✨ Features
* **BMI Calculator:** Instant BMI calculation tumhare weight aur height ke basis par.
* **Health Analytics Trend:** Ek interactive trend graph (chart) jo health data ko visually dikhata hai.
* **Report Download:** Ek click mein medical analytics report ko PDF format mein download karne ka option.

## 🛠️ Tech Stack Used
* **HTML5:** Structuring aur dashboard layout ke liye.
* **CSS3:** Gradient background aur modern UI styling ke liye.
* **JavaScript (JS):** Analytics logic, dynamic charts, aur PDF generation trigger karne ke liye.
